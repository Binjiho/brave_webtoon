* [Nodejs 설치](https://nodejs.org/ko/download)

* intellij 터미널(command Prompt)열기
* [ npm install -g @vue/cli ]
* [ vue --version ]
* [ npm init nuxt-app frontend ]
* [ cd frontend ]
* [ npm run dev ]
